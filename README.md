# E-commerce DataSets Dashboard
## Project Objective
A comprehensive e-commerce analytics dashboard built with vanilla JavaScript, Chart.js, and a dark-themed UI inspired by classic trading ledgers. This dashboard provides real-time visualization and filtering of order data from a morning trading session.

## KPI Strip — At-a-Glance Metrics
Six interactive metric cards display:

Total Revenue (with line-item count)

Order Volume (unique invoices)

Average Order Value

Units Sold (total quantity)

Customer Count (unique buyers)

Product Count (distinct SKUs)

Each KPI features smooth count-up animations when filters are applied, providing immediate visual feedback.
